## 🎯 项目简介

太一图表 Agent 是一个全域自进化的数据可视化智能体，融合全网设计趋势学习、图表类型自动推荐、可视化方案优化、递归进化，实现真正的自进化可视化智能体。

**核心能力**:
- 📊 全网设计趋势学习 (50-80 篇/日)
- 📈 图表类型自动推荐 (20+ 图表类型)
- 🎨 可视化方案优化 (+6%/代)
- 🔄 递归进化循环
- 📊 进化监控仪表板

## 🚀 快速开始

### 安装依赖

```bash
pip install -r requirements.txt
```

### 启动 Agent

```bash
# 启动 Agent
python3 core.py --start

# 执行学习循环
python3 core.py --learn

# 执行进化循环
python3 core.py --evolve --generations 100

# 显示仪表板
python3 core.py --dashboard

# 生成图表
python3 core.py --generate --type flowchart --data data.json
```

### Python API

```python
from core import DiagramAgent

# 创建 Agent
agent = DiagramAgent()

# 启动学习
agent.start_learning()

# 执行进化
agent.auto_evolve(generations=100, target_fitness=0.95)

# 生成图表
chart = agent.generate_chart(data, "flowchart")

# 推荐图表类型
chart_type = agent.recommend_chart_type(data)
```
